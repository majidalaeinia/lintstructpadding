# Lint Struct Padding

# Installation
```bash
git clone git@github.com:majidalaeinia/lintstructpadding.git
```

```bash
cd lintstructpadding
go build
```

### Run the Linter
```bash
./lintstructpadding ./examples/fileone.go
```

### Fix a Sample File
```bash
./lintstructpadding --fix ./examples/fileone.go
```
